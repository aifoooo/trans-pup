## TransPup 

TransPup 专注于提供流畅的英文网页翻译体验。

核心功能简洁高效：

- **划词翻译**：选中英文文本，即时显示翻译结果，无需切换页面；
- **翻译框翻译**：在翻译框中输入文本，快速获取准确翻译。

让 TransPup 帮你轻松理解英文内容，提升阅读效率～

## 如何使用

1. 环境准备
   - 安装 **pnpm**：`npm install -g pnpm`
2. 项目初始化
   - 安装依赖：`pnpm install` 或者 `pnpm install --registry=https://registry.npmmirror.com`
3. 开发与构建
   - 开发模式：`pnpm dev`
   - 生产打包：`pnpm build`
4. 扩展安装
   - 在 Chrome 浏览器中打开：`chrome://extensions`
   - 确保「开发者模式」已开启；
   - 点击「加载未打包的扩展程序」，选择 `dist` 文件夹，导入扩展程序。
